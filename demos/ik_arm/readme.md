# IK arm
Three-link arm for ik tests
Uses [URDF-VIS](https://github.com/openrr/urdf-viz)

### Dependenices 
    ```
    pyautogui

    json

    requests

    numpy
    ``` 
### How to launch

Fire up one terminal with following command
```    urdf-viz arm/urdf/arm.urdf
```

Then launch ``` controller.py ```
